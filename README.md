# Japanese Ruby PDF Generator

A desktop tool to convert Japanese text into Ruby (furigana) format and export as PDF.

## Features

- Automatic furigana generation (MeCab / UniDic)
- Adjustable font size and layout
- Real-time preview
- One-click PDF export
